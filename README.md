# entertainment-website

#### 介绍
一次程序设计大作业的前端代码，校园课外娱乐网，功能上与豆瓣有部分相似，读书、音乐、电影、户外活动，用户可以发起活动或参加活动，记录随笔等。 
前端技术栈：vue、vuex、vue-router、elementui、buefy-ui。
